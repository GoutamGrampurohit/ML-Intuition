# ML-Intuition

A collection of Machine Learning algorithms implemented in Python to develop a strong understanding of the mathematical concepts and implementation details behind classical and modern machine learning techniques.

The repository focuses on learning algorithms from first principles while also providing implementations using standard machine learning libraries for comparison and validation.

---

## Project Objectives

* Implement machine learning algorithms from scratch.
* Understand the mathematical intuition behind each algorithm.
* Compare custom implementations with established libraries such as scikit-learn.
* Build a well-documented reference for machine learning concepts and coding practices.
* Serve as a learning resource for interviews, academic study, and personal projects.

## Installation

Clone the repository:

```bash
git clone https://github.com/GoutamGrampurohit/ML-Intuition.git
cd ML-Intuition
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment.

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

---

## Running an Algorithm

Example:

